Welcome to CB-Manager documentation!¶

CB-Manager provides various APIs to interact with the Context Broker’s database. Through a REST Interface, it exposes data and events stored in the internal storage system in a structured way. It provides uniform access to the capabilities of monitoring agents.

Compatibility¶

cb-manager requires Python 3.5+.

Installation¶

You can install cb-manager with pip:

$ pip install cb-manager

or with easy_install:

$ easy_install cb-manager
